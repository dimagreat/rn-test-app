# React Native App

## How to Start:

1. Clone repository
```bash
git clone https://github.com/dimagreat/rn-test-app.git
```

2. Install dependencies

```bash
cd rn-test-app && yarn
```

3. Run

> Make sure that you have react-native-cli installed

- On Android

```bash
react-native run-android
```

- On ios

```bash
react-native run-ios
```


## Used dependencies:

* react-native
* redux
* redux-saga
